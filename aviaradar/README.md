# Userscripts for aviaradar.ru

## Установка

1. Установить плагин Tampermonkey: https://www.tampermonkey.net/?locale=ru
2. (только Chrome) My Extentions -> Tampermonkey -> включить опцию "Allow User Scripts" 
3. В Tampermonkey создать новый скрипт (например через Dashboard)
4. Заменить его содржание полностью этим скриптом
5. Добавить свои радары в список RADARS в начале скрипта (для координат можно воспользоваться, например, https://www.gps-coordinates.net/)

### Использование

1. Зайти на сайт своего радаза (радара из списка) - https://aviaradar.ru/top-radars/radar/779......
2. В разделе "Полярная диаграмма" появится ссылка "КАРТА"
